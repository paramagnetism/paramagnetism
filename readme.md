IP地址配置：
（1）服务端cmd命令控制行输入ipconfig，查询服务端IP地址
（2）复制服务端IP地址，将程序中所有IP地址（192.168.43.31）全部替换成服务端IP地址

客户端得先登录进去再点exit，不然服务端无法再起连接。
